          📐 Kumpulan Rumus SNI 3964:2025

          ① Proyeksi Penduduk (Geometrik)

          Pt = P₀ × (1 + r/100)ⁿ P₀ = penduduk tahun dasar | r = laju pertumbuhan (%/tahun) | n = selisih tahun

          ② SRT — Jumlah Contoh (Pasal 4.2.2)

          Alternatif 1: Jika Pt ≤ 1.000.000 → Ps = Cd × √Pt Jika Pt > 1.000.000 → Cj = Pt/1.000.000 ; Ps = Cd × Cj × √Pt Alternatif 2 (Slovin): n = N / (1 + N × e²) ; e ∈ {0.01, 0.05, 0.10} Jumlah KK : K = Ps (atau n) / N̄ Jumlah Rumah: Nr = Sn × K (per tingkat ekonomi H/M/L)

          ③ SSSRT — Jumlah Contoh (Pasal 4.2.3)

          Alternatif 1 : S = Cd × √Ts Alternatif 2 : n = N / (1 + N × e²) Ts / N = jumlah unit fasilitas sejenis di wilayah

          ④ TPS/TPS3R/TPST — Jumlah Contoh (Pasal 4.2.4)

          Jika N > 30 kali/hari → Alternatif 1: n = N / (1 + N × e²) Jika N ≤ 30 kali/hari → Alternatif 2: n = ceil(10% × N) (dibulatkan ke atas) N = total ritasi/kunjungan alat pengumpulan per hari

          ⑤ TPA — Jumlah Contoh (Pasal 4.2.5)

          Alternatif 1 : n = N / (1 + N × e²) Alternatif 2 : n = ceil(10% × N) N = total ritasi alat angkut per hari yang masuk ke TPA

          📌 Catatan Teknis Lapangan
          • SRT dihitung terpisah untuk setiap level ekonomi: High, Medium, Low.
          • SSSRT dihitung terpisah per jenis fasilitas (komersial, industri, fassos, fassum, fasilitas lain).
          • Frekuensi sampling: minimal 8 hari berturut-turut, atau minimal 3 hari (hari kerja, libur, puncak).
          • Sampling dilakukan pada musim hujan dan musim kemarau.
          • Asumsi: 1 rumah = 1 KK (Pasal 4.2.2).
          • Alat pengangkutan di TPA harus mewakili setiap jenis sumber/kawasan.
          • Sampling dilakukan sekurang-kurangnya setiap 5 tahun sekali.
